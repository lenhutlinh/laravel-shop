=== CẤU TRÚC BẢNG PRODUCTS ===

Table: products
Columns:
- id: ID sản phẩm (auto increment)
- shop_id: ID của shop bán sản phẩm (foreign key → shop)
- category_id: ID danh mục (foreign key → categories)
- subcategory_id: ID danh mục con (foreign key → subcategories)
- productName: Tên sản phẩm (string)
- price: Giá sản phẩm (float)
- description: Mô tả chi tiết (string)
- categoryName: Tên danh mục (string)
- subCategoryName: Tên danh mục con (string)
- previewImage: Ảnh đại diện (string, nullable)
- status: Trạng thái (1: đang bán, 0: ngừng bán)
- created_at: Ngày tạo
- updated_at: Ngày cập nhật

Table: products_variations_options
- variation_id: ID phân loại

- variationName: Tên phân loại (Màu sắc, Kích thước, Dung lượng...)
- variation_value: Giá trị (Đỏ, Size M, 256GB...)

- id: ID tổ hợp

=== SẢN PHẨM BÁN CHẠY ===

Top sản phẩm được tính theo:
- Tổng số lượng đã bán (từ bảng order_detail)
- Số lượt đánh giá
- Điểm đánh giá trung bình

Query mẫu:
SELECT products.id, products.productName, products.price, 
       SUM(order_detail.product_quantity) as sales_quantity
FROM products
LEFT JOIN order_detail ON products.id = order_detail.product_id
WHERE products.status = 1
GROUP BY products.id
ORDER BY sales_quantity DESC

=== TRẠNG THÁI SẢN PHẨM ===

status = 1: Đang bán (hiển thị trên trang chủ)
status = 0: Ngừng bán (không hiển thị, không mua được)

Sản phẩm bị ẩn khi:
- Shop tạm dừng bán
- Hết hàng (avaiable_stock = 0)
- Bị admin khóa do vi phạm

=== GIÁ SẢN PHẨM ===

Giá lưu dưới dạng: float (VNĐ)
Ví dụ: 
- 50.000đ → 50000
- 1.500.000đ → 1500000
- 28.990.000đ → 28990000

Hiển thị: Tự động format với dấu chấm ngăn cách hàng nghìn

=== TÌM KIẾM & LỌC SẢN PHẨM ===

Các tiêu chí lọc:
1. Theo danh mục (categoryName)
2. Theo danh mục con (subCategoryName)
3. Theo khoảng giá (price)
4. Theo shop (shop_id)
5. Theo từ khóa (productName, description)

Sắp xếp:
- Mới nhất (created_at DESC)
- Bán chạy (sales_quantity DESC)
- Giá cao → thấp (price DESC)
- Giá thấp → cao (price ASC)
- Phổ biến (views, ratings)
